Заявки с сайта → Google Таблица (простые шаги)

Откройте эту страницу в браузере и нажимайте ссылки по порядку. Почта для вопросов: dm.martyshenko@gmail.com

  1. Войти в Google. Если не залогинены — вход в аккаунт Google, почта Gmail mail.google.com
  2. Создать форму. Нажмите: docs.google.com/forms → создать новую форму (или forms.google.com → «Пустая форма»).
  3. Три вопроса в форме (кнопка «+» слева добавляет вопрос):
    1) Краткий ответ — «Имя и организация»
    2) Краткий ответ — «Email»
    3) Абзац — «Сообщение»
  4. Таблица для ответов. Вверху формы вкладка «Ответы» → зелёная таблица «Создать электронную таблицу». Справка Google: как связать форму с таблицей.
  5. (По желанию) Письма о новых заявках. Шестерёнка «Настройки» → «Ответы» → уведомления на dm.martyshenko@gmail.com.
  6. Ссылка для сайта. Кнопка «Отправить» (вверху справа) → вкладка <> Встроить. Как встроить форму (справка). Скопируйте из кода только адрес внутри src="..." (длинная ссылка на docs.google.com/forms/…).
  7. Вставить в сайт. Откройте в Cursor / Блокноте файл google-form-config.js в папке сайта. В строке embedUrl: "" вставьте ссылку между кавычками, сохраните файл. Пример:
    embedUrl: "https://docs.google.com/forms/d/e/....../viewform?embedded=true",
  8. Проверить. Откройте ваш лендинг в браузере и обновите страницу (F5). Должна появиться встроенная Google Форма. Отправьте тест — строка должна появиться в таблице.
